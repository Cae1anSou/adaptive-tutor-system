Cleaned questionnaire exports generated by `clean_questionnaire.py`.

- `participants.csv` – participant metadata (IDs, groups, demographics, prior experience, timestamp).
- `module_feedback_long.csv` – each row is a free-text response for a module/question-type pair.
- `scale_responses_long.csv` – tidy format for SUS, NASA-TLX, and self-efficacy items.

Regenerate via:

```bash
python Data_Analyze/questionnaire/clean_questionnaire.py
```
